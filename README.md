#README

## Library

```sh
npm install nodemon --save-dev
npm install eslint --save-dev

npx eslint --init
```

## Package

```json
    "scripts": {
      "start": "nodemon server.js",
      "lint": "eslint ./"
    },
```

## Run Server

```
npm run start
npm run lint
```
